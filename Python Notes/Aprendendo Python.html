Para solucionar esse erro precisamos trabalhar de trás para frente, olhando a saída do programa e tentando imaginar o que ele está fazendo. Nome de variáveis e palavras reservadas ~ Nomes de variáveis podem ser longos, porém os espaços devem ser completados com underline ( _ ), por exemplo: preco_do_cha. Eles podem conter tanto letras quanto números, mas têm de começar com uma letra. ~ Lembre-se que letras maiúsculas e minúsculas são diferentes. Pato e pato são variáveis diferentes. ~ muito$ é uma variável inválida por conter caractere ilegal. ~ Python tem 29 palavras reservadas nas quais não conseguimos criar variáveis, funções e etc. são elas: and def exec if not return assert del finally import or try break elif for in pass while class else from is print yield continue except global lambda raise Operadores e operandos ~ Operadores são símbolos especais que representam operações matemáticas. Os valores que o operador utiliza são chamados de operandos. ~ Os símbolos +, -, / e o uso de parênteses para agrupamento têm o mesmo significado que na matemática. O asterisco sozinho ( * ) é o símbolo para multiplicação e o aterisco duplo ( ** ) é o símbolo para potenciação. Ordem dos operadores ~ Quando mais de um operador aparece em uma expressão, a ordem de avaliação depende das regras de precedência. O acrônimo PEMDAS é uma prática de lembrar a ordem das operações: ~ P: Parênteses têm a mais alta precedência e pode ser usado para forçar uma expressão a ser avaliada na ordem que você quiser. Exemplo: 2 * (3 - 1) é 4. ~ E: Exponenciação ou Potenciação tem a próxima precedência mais alta. Exemplo: 3 * 1 ** 3 é 3 e não 27. ~ MDAS: Multiplicação e Divisão têm a mesma precedência, sendo elas mais altas que Adição e Subtração. Exemplo: 2 * 3 - 1 é 5 e não 4. ~ Operadores com a mesma precedência são avaliados da esquerda para direita. Adicionando novas funções ~ Função é uma sequência nomeada de instruções ou comandos, que realizam uma operação desejada. Esta operação é especificada numa definição de função. ~ A sintaxe para uma definição de função é: def NOME_DA_FUNCAO ( LISTA DE PARAMETROS ) : COMANDOS ~ Você pode usar o nome que quiser para as funções que criar. ~ A lista de parâmetros especifica que informação você deve fornecer para poder usar a nova função. ~ Uma função pode ter quantos comandos forem necessários, mas eles precisam ser endentados a partir da margem esquerda. ~ Parênteses vazios indicam que a função não tem parâmetros. Operador módulo ~ O operador módulo produz o resto da divisão do primeiro pelo segundo, ele é representado pelo símbolo de porcentagem ( % ). Exemplo: >>> quociente = 7 / 3 >>> print quociente 2 >>> resto = 7 % 3 >>> print resto 1 Expressões booleanas ~ Expressão booleana é uma expressão que retorna True ou False. Uma expressão que é verdadeira tem o valor 1, e uma que é falsa tem o valor 0. ~ Operadores de comparação: x == y x é igual a y x != y x é diferente de y x > y x é maior que y x < y x é menor que y x >= y x é maior ou igual a y x <= y x é menor ou igual a y ~ Existem três tipos de operadores lógicos, são eles: and, or, not. ~ Para que and seja verdadeiro é necessário que as duas operações sejam verdadeiras. Por exemplo: x > 0 and x < 10 é verdadeiro somente se x for maior do que 0 e menor do que 10. ~ Para que or seja verdadeiro é necessário que apenas uma das operações seja verdadeira. Por exemplo: x == 0 or y == 3 é verdadeiro se x for igual a 0 ou y for igual a 3. ~ Para que not seja verdadeiro é necessário que a operação seja falsa. Por exemplo: not ( x > y ) é verdadeiro se x for menor que y, ou seja , ( x > y ) deve ser falso. Execução condicional ~ As instruções condicionais nos dão a habilidade de checar condições e mudar o comportamento do programa de acordo com elas. A forma mais simples é usando o if. Por exemplo: if x > 0 print "x é positivo" Neste caso se x for maior que 0 o programa irá mostrar "x é positivo", caso x não seja maior o programa não irá apresentar nada. ~ Ocasionalmente é útil ter um corpo sem nenhuma instrução. Nesse caso podemos usar o comando pass, que indica ao Python: " passe por aqui sem fazer nada" Execução alternativa ~ A execução alternativa é um formato de instrução na qual existem duas possibilidades e a condição ( True or False ) determina qual será executada pelo programa. Para isso utilizamos o comando if junto do else. Por exemplo: if x % 2 == 0: print x, "é par!" else: print x, "é ímpar!" Neste caso se o resto da divisão de x por 2 for igual a 0 o programa nos mostrará que x é par, e se o resto da divisão de x por 2 não for igual a 0 ele nos mostrará que x é ímpar. Condicionais encadeados ~ Uma condicional encadeada é uma maneira de expressar uma operação com mais de dois branches. Para isso utilizamos os comandos if, elif e else. ~ O comando elif pode ser usado seguidamente quntas vezes quiser, contanto que o ultimo comando seja o else. Por exemplo: if x < y: print x, "é menor que" , y elif x > y: print x, "é maior que" , y else: print x, "e" , y, "são iguais" Neste caso se x for menor que y o programa executará o primeiro comando, se x não for menor que y o programa irá passar para o elif verificando se x é maior que y, caso x seja maior que y o programa irá executar o segundo comando, se x não for maior que y o programa irá executar o terceiro comando, no caso o else, dizendo que x e y são iguais. ~ As condicionais também podem ser agrupadas dentro das outras. Por exemplo: if x == y: print x, "e", y, "são iguais" else: if x < y: print x, "é menor que", y else: print x, "é maior que", y ~ Essas condicionais agrupadas são chamadas de condicionais aninhados. ~ O comando return permite terminar a execução de uma função. Exemplo: import math def imprimeLogaritimo ( x ) : if x <= 0: print "Somente números positivos, por favor" return resultado = math.log ( x ) print "O log de x é ", resultado Neste caso se x for menor ou igual a 0 o programa irá mostrar a mensagem de erro (escrita no print) e retornar ao inicio da função. Recursividade ~ Uma função pode chamar a si própria dentro dela. Por exemplo: def contagemRegressiva ( n ) : if n == 0: print "Fogo!" else: print n contagemRegressiva ( n - 1 ) Quando chamarmos a função passando um número ( n ) o programa irá fazer a contagem até o 0 e exibirá "Fogo!". Desenvolvimento incremental ~ Comece com um programa que funcione e faça pequenas mudanças incrementais. Em qualquer ponto do processo, se houver algum erro, você saberá exatamente onde ele está. ~ Use variáveis temporárias para manter valores intermediários de modo que você possa checá-los ~ Uma vez que o programa funciona, você pode querer remover algum sacffolding, mas cuidado para não tornar o programa difícil de ser lido. O comando while ~ Aqui está como fica contagemRegressiva com um comando while: def contagemRegressiva ( n ) : while n > 0: print n n = n - 1 print "Fogo!" ~ Você quase pode ler o comando while como se fosse Inglês. " While n for maior do que 0, siga exibindo o valor de n e diminuindo 1 do valor de n. Quando chegar a 0, exiba a palavra Fogo!". Variáveis locais ~ Variáveis criadas dentro de uma definição de função são locais. Você não pode acessar uma variável local de fora da função em que ela "mora". Isto significa que você é livre para ter múltiplas variáveis com o mesmo nome, desde de que elas não estejam dentro da mesma função. Função Range ~ A função range pega argumentos e devolve uma lista ~ Com argumento simples ela cria um lista que inicia no 0 Exemplo: range (5) [0, 1, 2, 3, 4] ~ Com dois argumentos ela cria uma lista com todos os inteiros do primeiro ao segundo argumento, incluindo o primeiro mas não incluindo o segundo na lista Exemplo: range (1, 5) [1, 2, 3, 4] ~ Com três argumentos a função range cria uma lista igual a com dois argumentos, porem o terceiro argumento especifica o tamanho do passo. Exemplo: range (1, 10, 2) [1, 3, 5, 7, 9]
